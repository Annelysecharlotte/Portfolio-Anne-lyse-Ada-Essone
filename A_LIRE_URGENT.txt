MODE D EMPLOI RAPIDE

1) Copie les dossiers images/ et documents/ dans ton dossier du portfolio.
2) Si Windows demande de remplacer des fichiers, accepte.
3) Verifie que les noms sont exactement les memes dans ton code HTML.
4) Dans le terminal VS Code :
   git add .
   git commit -m "ajout images et documents"
   git push
5) Recharge le site avec CTRL + F5.
