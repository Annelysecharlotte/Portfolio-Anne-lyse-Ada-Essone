Sujet : Les bases de données dans le Cloud

Pourquoi ce sujet ?

J’ai choisi ce sujet car les bases de données sont essentielles dans le développement d’applications modernes. Lors de mon stage chez My GoodLife, j’ai utilisé Azure Data Studio, ce qui m’a permis de mieux comprendre l’importance de la gestion des données.

Définition

Une base de données dans le Cloud permet de stocker, gérer et sécuriser des données sur des serveurs distants accessibles via Internet. Elle évite aux entreprises de gérer elles-mêmes toute l’infrastructure matérielle.

Azure SQL

Solution Microsoft permettant de gérer des bases de données SQL dans le Cloud. Elle est adaptée aux applications professionnelles et aux environnements Microsoft.

AWS RDS

Service proposé par Amazon Web Services pour gérer plusieurs moteurs de bases de données comme MySQL, PostgreSQL ou MariaDB.

Google Cloud SQL

Service de Google permettant d’héberger des bases relationnelles dans le Cloud avec sauvegardes automatiques et haute disponibilité.

Avantages

  • Accès aux données depuis n’importe où
  • Sauvegardes automatiques
  • Meilleure sécurité des données
  • Évolution facile selon les besoins de l’entreprise
  • Réduction des coûts matériels

Limites et risques

  • Dépendance à Internet
  • Coût pouvant augmenter selon l’utilisation
  • Besoin de sécuriser les accès
  • Protection des données personnelles à respecter

Méthodes de veille utilisées

Pour suivre ce sujet, j’utilise principalement des sites spécialisés, la documentation officielle des fournisseurs Cloud, des articles techniques et des vidéos de formation.

Microsoft Learn AWS Documentation Google Cloud Articles techniques YouTube

Conclusion

Les bases de données dans le Cloud représentent une évolution importante pour les entre prises. Elles permettent de gagner en flexibilité, en sécurité et en performance. Ce sujet est directement lié à mon parcours en BTS SIO SLAM et à mon intérêt pour les bases de données.
